# Bee Language

Bee is a general purpose computer language.

## goals

Bee is a experimental project with original syntax aiming these goals:

1. safe;
2. efficient;
3. reliable;

## design 

Bee is going to be a rule based programming language.

1. strongly typed;
2. modular;
3. compiled;
4. open source;
5. feature reach;
