What is Bee?

Bee is an experimental, computer language designed by Sage-Code Laboratory started in June 2008. Original goal was to make a simple language for learning how to code. We disagree a lot over the design choices and the initial team has split. We have resume the design in 2022 with fresh ideas.

Bee Specification

Next we will describe Bee language specification in details using examples of code. This is a high level requirement documentation. We organize the pages in logical order from basic concepts to advanced to learn the language syntax as if the language would bne functional. We assume you will read this documentation many times to fully understand all the examples before you can contribute to project design.

Disclaimer: In this documentation we refer to Bee Language using perfect tense like it would be already realized, but this language has no implementation. Please read instead of "is", "it will be". We do this to simplify the text and avoid future modifications.

Bee Keyboard

Bee require Unicode symbols so it is hard to type. To overcome this difficult obstacle we have design a special keyboard. One day maybe we will manage to create this keyword to fix productivity issues. With this keyboard Bee is not an awful language but good. We will have other technical solutions to improve typing but this would be the best.
